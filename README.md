## 📘 About EvidenceEngineStudio

**EvidenceEngineStudio** is a local desktop research workspace designed to bring scientific reasoning, evidence review, quantitative analysis, critique, and manuscript drafting into one structured environment.

The app is built to support a connected workflow instead of splitting research across disconnected tools.

With EvidenceEngineStudio, you can:

- create and organize research projects
- define and refine research questions
- build and inspect hypothesis structures
- link papers and evidence to a project
- import and inspect datasets locally
- run statistical analyses and review outputs
- critique hypotheses, sources, and data before drawing conclusions
- draft manuscripts with sections, figures, data attachments, citations, and formulas
- export manuscript drafts for further development

EvidenceEngineStudio is intended as a **research and writing environment**. It is not a medical device and not a substitute for scientific judgment, peer review, institutional policy, or expert supervision.

---

## 🧠 Evidence Layer / EvidenceEngine

EvidenceEngineStudio can be linked to **EvidenceEngine** as an external evidence search layer.

EvidenceEngine is intended to support literature retrieval and evidence exploration while the main project structure, writing, and research workflow remain local in the app.

### 🌍 EvidenceEngine
**Website:** https://evidenceengine.eu

EvidenceEngine is described as:

- AI-assisted
- European
- hosted in Europe
- privacy-focused

EvidenceEngineStudio itself is designed as a **local-first application**.  
This means the project workspace is intended to run locally on the computer, while online connectivity is only required when the user actively uses the external evidence layer via **evidenceengine.eu**.

## 🔐 Privacy

EvidenceEngineStudio is designed for local scientific work.

In general, the intended workflow is:

- local project organization
- local manuscript drafting
- local critique and reasoning workflow
- local dataset handling
- optional online evidence retrieval through EvidenceEngine

### User responsibility

If you work with sensitive or regulated data, you are responsible for ensuring that your use of the app complies with:

- GDPR
- institutional data protection requirements
- local research rules
- ethics and governance requirements
- clinical or academic confidentiality standards

Please avoid uploading or sharing identifiable personal data in GitHub issues, screenshots, or support emails.

Whenever possible:

- anonymise datasets
- remove direct identifiers
- use synthetic test examples for debugging
- check institutional policy before processing sensitive material

## ⚠️ Research / Scientific Disclaimer

EvidenceEngineStudio is intended to support scientific organization, analysis, critique, and manuscript development.

It is **not** intended to:

- replace scientific judgment
- provide binding medical or legal conclusions
- guarantee correctness of analyses or literature interpretation
- replace peer review
- replace institutional supervision or methodological consultation
- automate final scientific decision-making

All outputs, critiques, manuscript drafts, and AI-assisted suggestions must be reviewed critically by the user.

You remain responsible for:

- verifying sources
- verifying analyses
- checking statistical assumptions
- interpreting results appropriately
- ensuring that final claims are justified by the evidence
